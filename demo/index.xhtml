エラーが発生した場合はJSONファイルが表示 (またはダウンロード) されます。

CSV (カンマ区切りの値) ファイルはExcelに関連付けられている場合がありますが、 これはMicrosoft Officeのバグのようなもので、実際はExcelでCSVファイルを開くことはできません (正確にはRFC4180に準拠したCSVファイル)。

CSVファイルを開く際は、LibreOffice (またはApache OpenOffice) の 表計算ソフト「Calc」など、CSVに対応したソフトをご利用ください。 Microsoft Officeをインストールしている場合は、 前述のようなのソフトをインストールする際に *.docx や *.xlsx などの関連付けが上書きされないようにご注意ください (インストール時に「カスタム」を選び「ファイルの種類」でチェックをすべて外す) 。
