%から始まる行はコメントです
%
%クイズ問題データは、
%問題形式
%解答形式
%の組で記述していきます。
%
%基本的に、全ての数字やオプションなどはカンマ「,」区切りです。

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%問題形式の書き方
%Q,出題の種類,問題文[,問題オプション,問題オプション,・・・]
%
% 出題の種類
%  0 ・・・普通の文章問題です。文章のみを表示できます。
%　1 ・・・音声ファイルを再生することができます。問題文の次に音声ファイルを指定します。
%  3 ・・・Wikipediaからランダムに取得したページを元にクイズを作成します。
%  4 ・・・アンサイクロペディアからランダムに取得したページを元にクイズを作成します。
%
% 問題文
% 文章をそのまま入れます。これは解答などにも言えることですが、
% 文章中に「,」、「=」を使うことはできません
% 大文字にするなどして対処してください。
% また、\nで改行ができ、\wで少しだけウェイトします。
% \\と打つと\が表示されます。
%
% 問題オプションの種類
% start=数値　　　　音声ファイルをの再生開始位置を指定します。(単位はミリ秒)
% repeat=数値 　　　音声ファイルをの々回数を指定します。
% length=数値 　　　音声ファイルの再生長を指定します(単位はミリ秒)。-1だと最後まで再生します。
% speed=数値　　　　音声ファイルの再生速度を指定します。100が通常の速さです。
% score=数値　　　　この問題の得点を設定します。省略時はデフォルトのスコアになります
% finalscore=数値   最終得点を設定します。得点は時間がたつにつれて、最終得点に近づきます。
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%解答形式の書き方
%A,解答の種類,解答1[,解答2,・・・][,解答オプション,・・・]
%
% 解答の種類
% 0 ・・・　記述形式。解答に設定した言葉を答えさせます。
%       言葉の直後が　[[ の場合、その言葉で始まる言葉が正解に、
%　　　 言葉の直後が  || の場合、その言葉が含まれる言葉が正解になります。
%また、しりとりと同様に、[ ,|, ], を使ってプレフィックスやサフィックスを設定できます
% [,以後はプレフィックス(空でもOK)
% |,以後は本体
% ],以後はサフィックス(空でもOK)
%全ての組み合わせが登録されます。(何もつけてない場合は全部本体です)
%例えば
%A,0,[,,青い,|,空,],,と海
%と記述した場合、
% 青い空と海,青い空,空と海,空
% の４種類が解答として登録されます。
%

% 1 ・・・　選択形式。正解の選択肢の数字を選ばせます。
%　　　 正解にしたい選択肢の場合は、直後に「\seikai」を入れてください。
%       正解を複数設定することもできます。

% 2 ・・・　並べ替え形式。選択肢の数字を正しい順序で答えさせます。
%　　　 解答の順番については、言葉の直後に順番を指定します。
%       順番は1から始まる数字で、選択肢の数だけ漏れなく入れてください。
%
% 3 ・・・　全選択形式。当てはまる数字を全て選ばせます。
%　　　　選択形式の場合と同様、正解にしたい選択肢の直後に「\seikai」を
%        入れてください。
%
%なお、解答の全角は半角に、カタカナはひらがなに、大文字は小文字に直されますので、
%解答時はそれらは区別されません。
%
% 解答オプションの種類
% 
% \norandom          選択肢をシャッフルしなくなります
% \bonus=数値　　　　記述形式の場合、直前の言葉を答えて正解した場合のボーナス得点
% \explain=文字列    問題終了後に表示される解説を設定できます。
%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

%以下問題のサンプル
%QとAの間には空行があってもかまいません。

Q,1,何の曲でしょう？,C:\Documents and Settings\俺だよ俺\さんぷる.mp3,start=0,length=-1,repeat=0
A,0,サンプル,\bonus=5,\explain=音楽ファイルは絶対パスでも相対パスでもＯＫです。ただし、ファイルがないと出題時にエラーとなります。

Q,0,こたえは「あ」で始まる言葉。もしくは「お」が含まれる言葉
A,0,あ,[[,お,||,\explain=ここに解説を書く

Q,0,こたえはう\w\w\w\w\nとみせかけて、こたえはえ
A,1,あ,い,う,え,\seikai,お,か,\explain=選択肢の数は上限はありませんが、表示の関係上、問題文のことも考えると8つぐらいまでが限界だと思います。\nまた、\\wは複数続けて記述することで時間を調整できます

Q,0,春から順に並べてください
A,2,春,1,夏,2,秋,3,冬,4,\explain=並べ替え問題は、選択肢の後の順番を忘れずに入れましょう
Q,0,ひらがなをすべて選べ
A,3,ぼく,\seikai,俺,わし,\seikai,Me,\explain=正解の選択肢が少なくともひとつは必要です
Q,0,素数をすべて選べ
A,3,3,\seikai,234,127,\seikai,831,\explain=正解の選択肢が少なくともひとつは必要です

Q,0,「隣の客はよく柿食う客だ」と入力してください
A,0,隣の客はよく柿食う客だ,\explain=タイピングゲームっぽくすることもできます
